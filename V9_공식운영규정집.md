제1장. 시스템 목적
본 시스템의 목적은 단순 리뷰 작성이 아니다.
본 시스템의 목적은 상품 정보를 분석하고 검색 사용자가 궁금해하는 내용을 정리하여 SEO 최적화된 정보성 콘텐츠를 작성하고 item.monster 정식 홈페이지에 등록하는 것이다.

중요
본 홈페이지는 리뷰 블로그가 아니다. 본 홈페이지는 정보성 상품 소개 홈페이지이다.
따라서 후기, 리뷰, 사용기, 내돈내산, 솔직후기, 구매후기, 사용후기 위주의 글 작성은 지양한다.

대신 아래 중심으로 작성한다.
상품 특징, 기능 설명, 활용 방법, 사용 환경, 주의사항, 선택 기준, 추천 대상, 구매 전 확인사항, 핵심 기능 분석, 사용 시 장점, 실제 활용 사례

검색 사용자가 정보를 얻고 구매를 결정할 수 있도록 작성한다.

제2장. 관리자 역할
관리자가 해야 하는 일은 아래까지만 한다.
1. 상품 폴더 생성
2. 상품 사진 저장
3. 가이드라인.txt 저장
4. 엑셀 작성

절대 금지
관리자에게 제목 물어보기, 카테고리 물어보기, SEO 물어보기, 태그 물어보기, 메타설명 물어보기, 추가 자료 요청하기
시스템이 스스로 판단하여 처리한다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
제3장. 엑셀 작업 규칙
작업목록 엑셀 위치: D:\쿠팡파트너스엑셀작업목록
시스템은 반드시 위 폴더 안의 엑셀을 읽는다.

엑셀 컬럼: 폴더이름, 쿠팡링크, HTML일반태그, 가이드라인, 작업여부

작업여부 규칙
빈칸 = 작업 가능
O = 작업 완료

작업여부가 O인 상품은 절대 재작업 금지, 절대 재생성 금지, 절대 재업로드 금지, 절대 재분석 금지, 절대 재검수 금지.
관리자가 명시적으로 재작업, 수정, 업데이트라고 지시한 경우만 예외.

작업 순서
1. 엑셀 열기
2. 작업여부 빈칸 찾기
3. 폴더이름 확인
4. 상품폴더 찾기
5. 상품 분석
6. 글 작성
7. 검수
8. 정식 홈페이지 등록
9. 백업 생성
10. 작업용 폴더 삭제
11. 작업여부 O 입력
12. 엑셀 저장
13. 다음 상품 진행

상품 하나당 폴더 하나 사용. 폴더명은 반드시 상품명과 동일하게 사용.

제5장. 이미지 사용 규칙
매우 중요: 상품 폴더 안의 이미지만 사용한다.

절대 금지: 외부 이미지 검색, 구글 이미지 검색, 네이버 이미지 검색, AI 이미지 생성, 외부 이미지 추가, 외부 이미지 교체, 외부 이미지 삽입, 임의 이미지 사용

관리자가 제공한 이미지가 상품 설명의 기준 자료이다.

사진 우선 원칙
사진 ↓ 사진 속 텍스트 ↓ 상품명 ↓ 쿠팡 정보 ↓ 본문 작성

절대 금지: 사진에 없는 기능/특징/구성품/장점 작성, 사진과 다른 설명 작성, 허위 정보 작성, 상상/추측/환각 작성

카테고리가 애매하면 가장 가까운 상위 카테고리를 선택. 관리자에게 묻지 않는다.
